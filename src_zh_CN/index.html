界面开发优化
Dart 是专门围绕用户界面创建的编程语言
高效
可持续开发
反复修改：使用热重载在运行的应用程序中立即查看修改后结果
全平台
流畅运行
针对移动，桌面和后端应用编译为 ARM 和 x64 机器代码。为 Web 应用编译为 JavaScript
针对界面开发优化
  • 成熟且完整的 async-await 配合基于 isolate 并发， 用于实现包含事件驱动代码的用户界面
  • 编程语言针对构建用户界面进行了优化，功能包括用于扩展集合的 Spread 操作符 和用于为每个平台自定义用户界面的 Collection If
  • 编程语言语法亲切，易于学习
高效可持续开发
  • 反复的修改源代码，使用 热重载 在运行的应用程序中立即查看修改后结果
  • 使用灵活的类型系统编写代码，系统具有丰富的静态分析以及强大的 可配置工具
  • 在你使用的代码编辑器中进行 性能分析， 记录日志， 和 调试
全平台流畅运行
  • AOT-编译 将应用程序编译为本机机器代码，以便快速启动
  • 使用完整，成熟，快速的JavaScript编译器定位Web 针对 Web 提供完整，成熟，快速的 JavaScript 编译器
  • 使用同一种语言，可以编写支持你前端应用的 后端代码
